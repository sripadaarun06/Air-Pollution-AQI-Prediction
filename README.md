# Air Pollution AQI Prediction API

This project predicts Air Quality Index (AQI) using machine learning models and provides real-time inference via a FastAPI-based REST API.

## Features
- Multiple ML model comparison
- Explainable AI using SHAP
- Production-ready API
- Risk level classification

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- FastAPI
- SHAP
